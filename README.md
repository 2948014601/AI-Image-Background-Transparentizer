### 📌 项目简介

**PNG 背景透明化工具** 是一个基于 Python + PySide6 开发的桌面应用，用于将AI生成的 PNG 图片中的灰白棋盘格背景、纯白背景或浅色背景快速处理为透明背景，便于科研绘图中图标生成、PPT制作过程中插图生成等。

适合用于处理：

- 带灰白棋盘格背景的 PNG 图片
- 带纯白背景的图片
- 从网页、设计软件、截图中保存下来的伪透明图片
- 需要快速导出透明 PNG 的素材

---

### ✨ 功能特点

- 🎨 现代化无边框 UI 界面
- 🖱️ 支持拖拽导入 PNG 图片
- 🧩 支持灰白棋盘格背景去除
- ⚪ 支持纯白背景去除
- 🎚️ 支持阈值、灰阶范围、容差、边缘柔化等参数调节
- 🚀 内置多种处理模式：
  - 灰白方格
  - 纯白背景
  - 强力清除
  - 保守清除
- 👀 支持原图与处理结果预览
- 🔍 支持预览缩放、适应窗口、100% 显示
- 💾 支持导出透明 PNG
- 📦 支持 PyInstaller 打包为 Windows EXE

### 📁 使用方法

1. 双击exe启动程序。
2. 拖入 PNG 图片，或点击导入区域选择图片。
3. 选择合适的处理模式：
   - 灰白方格：适合棋盘格伪透明背景
   - 纯白背景：适合白底图片
   - 强力清除：适合背景较复杂或残留明显的图片
   - 保守清除：适合主体颜色较浅，避免误删
4. 根据需要微调参数。
5. 点击 **生成预览**。
6. 确认效果后点击 **导出 PNG**。

------

### ⚙️ 参数说明

| 参数             | 说明                                   |
| ---------------- | -------------------------------------- |
| 白色方格阈值     | 控制浅色背景识别范围，数值越低识别越多 |
| 灰色方格最小亮度 | 棋盘格灰色区域识别下限                 |
| 灰色方格最大亮度 | 棋盘格灰色区域识别上限                 |
| 灰阶容差         | 控制 RGB 接近灰色的判断范围            |
| 边缘柔化         | 减少主体边缘残留的灰白边               |
| 去除纯白背景     | 是否额外处理纯白背景                   |
| 纯白背景阈值     | 控制纯白背景识别强度                   |

------

### ⚠️ 注意事项

- 本工具主要用于去除浅色背景、灰白棋盘格背景和纯白背景。
- 如果图片主体本身包含大量浅灰色或白色，建议使用 **保守清除** 模式。
- 处理复杂背景时，可能需要手动调整参数。
- 输出格式为 PNG，以保留透明通道。
